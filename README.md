## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

- Update your name in `src/components/Hero.js`
- Add your projects in `src/components/Projects.js`
- Modify skills in `src/components/Skills.js`
- Add your social links in `src/components/Contact.js`

## Build for Production

```bash
npm run build
```
